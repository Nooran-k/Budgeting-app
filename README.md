# 💰Budgeting App

##  Description
This is a desktop budgeting application built using Java Swing. The application helps users manage their personal finances by tracking income, recording expenses, and organizing transactions into categories.

It is designed for students and individuals who want a simple, offline tool to monitor spending and improve budgeting habits.

---

## 🚀 Features
- User registration and login system
- Create/edit budget
- Add transaction
- Categorize transactions (Food, Transport, Bills, etc.)
- Budget over-limit Alert
- Set and track financial goals
- view reports and analytics

---

## 🛠️ Tech Stack
- **Language:** Java
- **GUI:** Java Swing
- **IDE(s):** IntelliJ IDEA, Visual Studio Code
- **Database:** SQLite (via JDBC)
- **Other Tools:** Git
